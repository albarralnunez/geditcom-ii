Empty Format

This is format has the minimum templates to run as a format. It could be used as a starting point for writing an entirely new type of format for GEDitCOM II using the GEDitCOM Editor. As is, all formatting is done by the default templates (the "_Unknown_ Record Editing" for records and the "_Unknown Editing" for all subordinate structures). These templates simply display the raw GEDCOM data with line levels indented for clarity.

Others things included in this format are:

  1. All required editing colors, fonts, and properties.
  2. A complete set of GEDCOM events and attributes for individuals and families. These lists include a few common custom evens using EVEN# tags.
  3. A common set of language keys for English strings including keys for all GEDCOM tags and records.
  4. This simple documentation page linked to a help.css style sheet file that matches the help style used in GEDitCOM II.
