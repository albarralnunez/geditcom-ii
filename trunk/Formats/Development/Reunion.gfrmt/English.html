Book Format

This format displays the text is a very printer friendly way. You can edit many parts of the data, but it has fewer editing options than the "Default Format".

All data is loaded into a single window without the use of child windows. In fact even linked data (such as notes and multimedia objects) are all loaded into the same window as well. While this approach is desireable for printing the entire content of records, it can result in long displays. In very long records, you may even notice a delay while all the data is formatted for display.

Book Format Features

  • Although there do not appear to be any editing fields, much of the text can be edited. Click the mouse in any text to see if editing is allowed. If you get a flashing cursor, you can edit the text.
  • You can drag and drop records on records to attach data as explained in the GEDitCOM II help.
  • You change the format's fonts and colors using the GEDitCOM II preferences.
  • The custom options in the "Editing" section of the GEDitCOM II preferences are:
    • Hang Indent: Set the left margin for the section labels within each record.
    • Left Margin: Sets the margin for text under the section labels and it is independen of the Hang Indent setting.
    • Level Indentation: Subordinate data is indented further from the root level data. Use this setting to select the indentation per level.
    • Multimedia Full Size: Set the maximum height for multimedia objects. Multimedia objects will always be scaled (if needed) to fit in the display area, but can be ruther limited to this size.
