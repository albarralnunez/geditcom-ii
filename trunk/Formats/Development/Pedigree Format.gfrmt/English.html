Pedigree Format

The "Pedigree Format" displays each individual as a four-generation pedigree chart (individual up to grandparents; hint: you can resize the individual window to display the entire chart and that size will be remembered). You can click on links in this view to quickly navigate around the genealogy data in your file:

  1. Click on any chart cell to view the pedigree chart for that individual (note: a line to the right of any grandparent means additional ancestors are documented; the lack of a line means no more ancestors are known).
  2. Click on any family icon button to view a family group chart for the spouses above and below the button.
  3. To see descendants, click on any name in the list of "Children." (note: a "+" in front of any child means additional descendants of that child are documented; the lack of a plus means that child has no documented children).
  4. If the individual had more than one marriage, you click click on names in the "Other Spouses/Children" list (note: a "+" in front of any other child means additional descendants of that child are documented; the lack of a plus means that child has no documented children).

You cannot edit the pedigree chart. Instead, to edit all the details on an individual click various tabs in the button bar at the top of the window for "Events," "Multimedia," and "Other." While most things can be edited, a few options in the "Default Format" are not available. You can alway switch back to the "Default Format" for any editing needs not found in the "Pedigree Format."

Besides individual records, other record editing options are:

  • Family Records: These records will show as a concise family group chart. You can edit marriage date and place and attach or detach children. To edit more family details, click the "Events and Others" or "Multimedia" tabs at the top of the window. To edit more marriage details, open the "Events and Others" section and click the "info" icon after the marriage event.
  • Other Records: All other records are edited by methods similar to the "Default Format" and mostly documented in the GEDitCOM II help. The colors and fonts, however, use the style of the "Pedigree Format."

Embedded Scripts

The following embedded scripts provided extra functions when control clicking on some fields:

Name Editing Fields
Control click to change the case of the name (e.g., to all uppercase, title case, or surname only in uppercase).
Lat,Lon Editing Fields
Control click to open that latitude and longitude in a website called GeoHack. That web sites will have numerous links to map or view that location.

Format Fonts

To change format-specific fonts, open the GEDitCOM II preferences window and click on the "Fonts" tab. The "Pedigree Format's" fonts are:

Label Text
Prominent text labels.
Static Text
Lower-level text labels.
Editing Text
The text used in editing fields.
Button Text
The text for buttons.
Check Box Text
The text next to check boxes.
Popup Menu Text
The text used in pop-up menus.
Hyperlink Text
The text used for clickable links between records
Children Number
The font used for the numbering of children in the family group chart. The color is ignored and set to the chart's background color. This font is also used for the family group chart title. In this use, the size is ignored and set to 18 pt.
GEDCOM Extension Text
The "Default Format" has some extensions to GEDCOM 5.5. As long as you are using GEDitCOM II, those extension will always work fine. If the export to another program, the extensions will exported, but the other program will probably not recognize the data is may even delete it. To help you recognize GEDCOM extension, these labels are indicated in the font. The custom tags are mostly in Source and Multimedia Object records.

Format Colors

To change format-specific colors, open the GEDitCOM II preferences window and click on the "Colors" tab. The "Pedigree Format's" colors are:

Window Background
The main color for the background of a browser window.
Record Header Box Background
The top section of the window background in a browser window uses this color rather than the main window background color.
Editing Background
The background for cells used for text editing.
Table Cell Background
The background for header cells used in tables of data.
Main Individual Background
The background color for the individual's cell in the pedigree chart window. You can click this cell to edit the individual and thus it is good for this color to differ from the background color.
Family Chart Background
The background color for family charts. This color is ignored when printing, which will use white background instead.

Format-Specific Variables

To change format-specific variables, open the GEDitCOM II preferences window and click on the "Editing" tab. The margin settings and default settings are common to all formats. The variables specific to the "Pedigree Format" are:

Multimedia Full Size
Images in multimedia records will display at the full size of the image in the file (which can be scaled using the scale menu). The larger dimension of the image, however, will never exceed the window width or this "Multimedia Full Size" setting, whichever is smaller. You can use the "Zoom" pane view the image at a scale larger than the window (as set with the scale menu).
Thumbnails Per Row
Links for records to multimedia objects are displayed with a thumbnail image to the object. This preference sets the number of thumbnails per row in the display. The thumbnails will also adjust in size if the window size is changed.
    To spped up display of thumbnails, GEDitCOM II generates small thumbnails for each multimedia object containing an image. These thumbnails look fine when the links are small. If the links get large (e.g., when using one thumbnail per row), this format will use the full scale image rather than the internal thumbnails. The image quality will be good, but it might slow down access to records with lots of linked images. To maximize speed when opening such records, it is best to increase this setting to at least 2 and you may need 3 or more if using a large monitor with large record windows.
Notes Position
A numeric code for location of the editing pane in Notes records. The options are: 1 = left, 2 = top, 3 = right, and 4 = bottom.
Chart Box Width
The width of cells in the pedigree chart. Make it large enough for your longest names.
Chart Box Width
The width of cells in the pedigree chart. Make it large enough for your longest names.
Chart Column Spacing
The horizontal spacing between generations in the pedigree chart
Chart Row Spacing
The vertical spacing between spouses in the pedigree chart
Chart Box Padding
Extra padding within the cell in the horizontal direction
